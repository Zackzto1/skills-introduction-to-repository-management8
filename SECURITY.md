# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them via email to:
techsupport@mergingtonhigh.example

Include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Response Timeline

- Acknowledgement within 2 school days
- Initial assessment within 5 school days
- Critical issues handled immediately
